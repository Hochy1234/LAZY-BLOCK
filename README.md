# LAZY-BLOCK
你做為一名專精程式工程師 你將幫助我完成此項目 我們使用 Python/c++/Rusty
並且UI使用Ttkboostrap
而我們要製作的項目名為 Lazy Block

容我詳細地為你介紹這個懶人文字產生器，使用者可以使用自己創建的方塊來產生已經"設置"好的文字。

介面上面會有一行來選擇工具列的"分類" 這個高度不會很大 差不多占16分之1
目前有的分類分別是
================================================================
          主要		功能		美術		其他
================================================================

而下面一行則是工具列 會依照所選的"分類" 高度約是16分之3
來決定會有那些工具 而最右邊固定是"創建"


再分類 美術裡面會有一個選項 用來更改 ttk的Window Theme
可以在裡面選擇所有的Window Theme

接下來是我們的重點區域

其餘的地方被分成3塊 我們暫且用 A B C區來表示
===================================================================
		|			|
		|			|
		|			|		
		|			|
	A	|		B	|	C		
  		|			|			
		|			|
		|			|
		|			|
		|			|
		|			|
		|			|
		|			|
====================================================================

A區: A區為放置方塊的地方 創建完後的方塊會在這裡 可以拖曳至B區來做使用
	A區上方會有一行可以用於選擇資料夾 而這個資料夾會是存放"方塊"(代碼檔)的地方
	所以每切換另一個資料夾後 顯示的方塊會是所選資料夾檔案裡面擁有的"方塊"

B區: B區為輸入文字的地方 在這裡你可以進行打字 並將方塊拖曳至此區

C區: C區為轉換文字的地方 會依照方塊要"轉換"的文字 來進行轉換文字的事項

總結:方塊會在A區 點擊方塊後 方塊會出現在B區 同時每當你進行打字或放置方塊C區會將方塊轉換成你方塊要求"轉換"的文字

舉個例子: 我們現在有一個方塊 [升級]

[升級] 所"顯示"就是方塊上的文字:(升級)
而他"產生"的文字是Upgrade!
則以此方塊為範例 他會變成:


B區的文字為:
	1!2!3! ABCD 哈哈哈 [升級] 一筆一筆
	第二段

C區所轉換的文字就會為:
	1!2!3! ABCD 哈哈哈 Upgrade! 一筆一筆
	第二段
###############################################################
關於方塊:
這是一個方塊 他使用的UI類型是Button 
而"方塊"會是一個文字檔(或資料夾，檔案之類的)
每個"方塊"在產出之後都會有設定好的格式以及代碼
好讓它放在B區時"轉換"變成C區的文字

至於我們要如何新增方塊呢？我們需要點下工具列的創建來產生一個新的方塊，產生方塊的介面會是這樣的：
======================================

請為你方塊(資料夾)命名：
[輸入]

請輸入你要"顯示"的文字：
[輸入]

請輸入你要"轉換"的文字：
[輸入]

工具區(將這些工具移動到"顯示"與"產生"裡面):
-----------------------------------
{填入文字()} 

-----------------------------------
======================================
創建完成後會再你A區目前的資料夾裡面再新增一個資料夾 資料夾名字為你在創建時輸入的
資料夾裡面會有：
========================
方塊的代碼(顯示與轉換)
UI代碼
========================
當方塊被刪除時 也會連同方塊資料夾一起被刪除


而工具區的工具，通常也都會是按鈕，也就類似"方塊中的方塊"
而{輸入文字()} 是可以在B區輸入文字並且直接原封不動轉換至C區
所以當他在按鈕裡面時，他是給你輸入文本的地方

對 這就是一個幫助我們產生文字的文字懶人產生器
而在之後會有更多功能以及更多美化 但我們先將這些做出來即可
